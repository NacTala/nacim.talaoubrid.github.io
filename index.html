2026 · RESEARCH INTERNSHIP

Imitation Learning for Dexterous Manipulation

ISIR — Sorbonne University | Robotics, manipulation and learning-based control

Full description

Built a full data pipeline for robotic manipulation, with synchronized multimodal sensing (vision and tactile signals).
Trained and evaluated imitation learning policies for robotic control, and studied how different tactile representations influence learning performance. Benchmarked several imitation learning algorithms, including Diffusion Policy, ACT, and MLPbased approaches. Introduced a shared control algorithms between the learned model and the human teacher (teacher-in-the-loop), enabling learning strategies inspired by approaches such as DAgger.

Objectives

Develop autonomous manipulation capabilities for dexterous hand and investigate the role of tactile sensing in learning-based control.

Technologies used

  • Python
  • ROS 2
  • PyTorch
  • Imitation Learning (Diffusion Policy, MLP, ACT)
  • Tactile sensing systems

Human-Centered AI

Smart Glasses for Emotion Recognition

Assistive wearable combining computer vision-based emotion recognition and haptic feedback

Full description

Designed an assistive smart glasses system for visually impaired and autistic users, integrating a camera and an AI module to analyze the facial emotions of the interlocutor in real time. Developed a first functional prototype featuring a pneumatic haptic feedback system embedded in the temples of the glasses, where the intensity of detected emotions (e.g., anger, joy) is translated into proportional inflation of textured cushions to provide intuitive tactile feedback to the user.

Objectives

Create an assistive wearable system that translates facial emotion recognition into intuitive haptic feedback for real-world social interaction support.

  • Computer vision
  • Deep learning
  • Embedded systems
  • Pneumatic actuation
  • Haptic feedback systems

Robot Learning

DAgger for Robotic Imitation Learning

Dataset aggregation and expert comparison on the FetchReach-v4 robotic environment

Full description

Implemented the DAgger (Dataset Aggregation) algorithm on the FetchReach-v4 robotic environment to improve imitation learning performance under distribution shift. Compared three types of experts: human demonstrations, a PID-based analytical controller, and a pre-trained SAC reinforcement learning policy. Evaluated the influence of expert quality on the learned policy and compared Behavioral Cloning with DAgger-based approaches.

Objectives

Study the impact of expert supervision quality and evaluate dataset aggregation methods for robust robotic imitation learning.

Technologies used

  • Python
  • PyTorch
  • Imitation Learning
  • DAgger
  • Reinforcement Learning
